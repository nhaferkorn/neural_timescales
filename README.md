# Neural Timescales in task-based EEG
This repository contains the code for the estimation of neural timescales in EEG data.
We re-analyzed a data set that includes a manipulation of episodic memory in younger and older adults. The raw data (EEG and Behavioral) can be openly accessed on the Radboud Repository (https://doi.org/10.34973/5xbh-zd57).

## Project Goals
This project seeks to eprxamine the relationship between cognitive variables and neural timescales.
Specifically, we contrast timescales following high-distraction stimuli and low-distraction stimuli during the encoding phase.

## Repo Structure 
All analysis scripts can be found in the scripts_notebooks folder.
Helper functions can be found in the timescales_memory folder.
